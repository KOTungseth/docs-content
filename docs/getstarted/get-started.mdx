In this getting started, you'll create your first Elasticsearch project. You can then ingest, search, and explore your data in Elasticsearch.
To get started with Elasticsearch, follow these steps:

1. Set up
    - Create your project
    - Create an API key
2. Ingest
3. Search
4. Explore
    - Run an aggregation with the client
    - Create a dashboard
