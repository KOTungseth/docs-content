In this getting started, you will go through the creation of your first project. This will allow you to interact with your Elasticsearch solution to ingest, search, and explore your data.
To get started with Elasticsearch, follow these steps:

1. Set up
    - Create your project
    - Create an API key
2. Ingest
3. Search
4. Explore
    - Run an aggregation with the client
    - Create a dashboard
